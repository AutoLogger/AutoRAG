Installation
============

.. note::
   Full content lands in step 4 of the docs rollout. This page is a
   stub so the toctree resolves.

AutoRAG is distributed from GitHub, not PyPI. Install with::

    pip install "autorag[all] @ git+https://github.com/AutoLogger/AutoRAG@v0.6.0"

See :doc:`internals/extras-model` for which extra you need.
